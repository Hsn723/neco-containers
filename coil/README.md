Coil container
==============

This directory provides a Dockerfile to build a Docker container
that runs [coil][].

Usage
-----

### Run coilctl

```console
# For docker
$ docker run quay.io/cybozu/coil:0 /coilctl
```

For more information and usage, see [coil][] project page

[coil]: https://github.com/cybozu-go/coil
